gork
====

A text based dungeon crawler using ncurses.
Features procedurally generated overworld with villages and quests as well
as procedurally generated dungeons.

Dependencies:
-------------
- goncurses

Features:
---------
- Procedural world generation
- Field of View
- Path finding

Planned:
-----
- UI
- Items
- Quests
- Villages and roads
- Factions
